
## 前言

大概三年前，自学 Java 的时候，在印象笔记里记录了大概上千条关于编程的笔记，杂乱无章，看起来很繁琐，自我认为觉有这么多资料如果不归纳，不整理，对于自个儿来说，那还不是跟躺在垃圾桶的垃圾一样，没啥价值。于是决定抽业余时间，好好整理一下。

这些资料算是本人自学 Java 的一个学习历程，其中不乏自己学习 Java 时的自己的理解，也有为了一个问题在搜索引擎上翻江倒海，还有不少各位大牛写的好文章等等。
